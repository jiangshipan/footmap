User:1. 用户注册 ip + /user/reg?{username,password} GET/POSTparam: {username,password} 不可缺少或为空例：http://localhost:8080/user/reg?username=wangbing&password=123456    Response:{"code":1,"message":"用户名已经存在","data":null}2. 用户登陆 ip + /user/login?{username,password} GET/POSTparam: {username,password} 不可缺少或为空例:http://localhost:8080/user/login?username=fanxingwei&password=123456   Response:{"code":0,"message":"登陆成功","data":null}3. 用户退出 ip + /user/loginout   GET例:http://localhost:8080/user/logout   Response:{"code":0,"message":"退出成功","data":null}Map:1. 插入地图节点 ip + /map/addPlace?{currentTime, currentPlace}param: {currentTime, currentPlace} // currentTime格式为 YY-MM-DD HH-MM-SS 例如：2019-3-16 10:40:27例: http://localhost:8080/map/addPlace?currentTime=2019-3-16 10:40:27&currentPlace=西安    Response: { "code": 0, "message": "新增成功", "data": null}2. 返回当天地图 ip + /map/getUserMap?{selectTime}param: {selectTime} 格式要求 2019-02-22 字符长度不得大于10例:http://localhost:8080/map/getUserMap?selectTime=2019-3-5    Response: {                  "code": 0,                  "message": "获取成功",                  "data": [                      {                          "currentTime": "2019-03-05T09:40:27.000+0000",                          "currentPlace": "西安"                      },                      {                          "currentTime": "2019-03-05T10:40:27.000+0000",                          "currentPlace": "北京"                      },                      {                          "currentTime": "2019-03-05T12:40:27.000+0000",                          "currentPlace": "西安"                      }                  ]              }
